<objective>
Build the trigger/heuristic layer (ANTIC-01) of the anticipatory-prompting SPIKE: a read-only
observation surface over a fixture-shaped chat+canvas state snapshot, a typed candidate contract,
and 2–3 deterministic (no-ML) triggers that each PROPOSE a structured `AnticipatoryCandidate` —
never fire anything. Ships behind a global feature flag that defaults OFF (D-12).

Purpose: Establish the SPIKE's contracts and observation layer first (interface-first), so the
Plan 25-02 gate chain and the Plan 25-03 findings harness operate on one candidate shape and one
snapshot shape. This is real-but-dark code (D-01), exercised against scripted fixtures (D-02).

Output: Feature flag + spike settings, the `AnticipatoryCandidate`/`AnticipatoryStateSnapshot`/
`AnticipatoryLifecycleEvent` frozen contracts, three scripted state fixtures, three deterministic
triggers, and the tests proving each fixture drives its trigger and that the flag gates everything.
</objective>

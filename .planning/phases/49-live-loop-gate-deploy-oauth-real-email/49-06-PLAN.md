| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-49-06-02 | Information Disclosure | client secret committed to a repo file | mitigate | secret is Dashboard-only per env table; checklist grep-gated to contain no GOCSPX- literal; never prefixed NEXT_PUBLIC_ |

| T-49-06-05 | Denial of Service | GitHub rename breaks OIDC deploys | mitigate | user decides rename-with-companion-apply vs re-park at this gate — a recorded, informed choice, never a silent break |
| T-49-06-SC | Tampering | npm/pip/cargo installs | mitigate | no new packages — terraform + existing tooling only |
</threat_model>

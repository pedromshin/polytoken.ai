<objective>
Implement the application-level cost circuit breaker (STREAM-03, FOUND-3, D-20/21/22): a cost ledger
port + Supabase adapter over `chat_cost_ledger`, env-configurable per-turn/per-session/per-day caps
(defaults $0.50 / $2.00 / $5.00), a fail-closed pre-turn estimate gate, and a per-turn running-cost
check that signals mid-stream abort. Also close the known usage-capture gap (D-22): capture real
token usage in the genui generator + judge adapters instead of dropping it.

Purpose: a general budget ledger drawn on by chat (and later studio/agents) — not a chat-shaped
guard bolted beside the AWS alert. This plan de-risks the streaming agent (22-06/22-07): the breaker
is a fully-tested unit before it is wired into the turn loop.

Output: ledger port, breaker service (TDD), Supabase adapter, cap settings, DI, D-22 gap fix, tests.
</objective>

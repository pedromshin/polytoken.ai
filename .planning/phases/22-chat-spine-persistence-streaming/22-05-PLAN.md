<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → tRPC chat router | conversation ids, titles, model ids are untrusted input at the tRPC boundary |
| tRPC → Postgres (Drizzle) | queries must be parameterized and importer-scoped |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-16 | Tampering (SQL injection) | conversation queries | mitigate | Drizzle parameterized queries only; ids validated as z.string().uuid(); title length-capped |
| T-22-17 | Information disclosure | cross-importer conversation read | accept | single-shared-key local/sandbox posture (mirrors genui backlog 999.1); importer scoping applied where an importer context exists; note for real multi-tenancy |
| T-22-18 | Tampering | accidental/irreversible data loss on delete | mitigate | hard delete gated behind an explicit AlertDialog confirm (D-14); no auto-fire |
| T-22-19 | Denial of service | unbounded history/list payloads | mitigate | list + history row counts capped; Zod-bounded inputs |
</threat_model>

<objective>
Ship the streaming transport (STREAM-01, D-24) and progressive genui emission (STREAM-02, D-02): a
FastAPI SSE endpoint that serializes the RunChatTurn agent's run events as `text/event-stream`, and
the `emit_ui_spec` tool — offered only to genui-capable models (D-05) — whose partial-JSON tool-call
deltas stream so the client can render a partial tree. Client disconnect aborts the run keeping the
partial (D-25/D-15). Regenerate is a sibling SSE run (CHAT-04); stop is realized as disconnect (CHAT-03).

Purpose: turns the tested agent generator (22-06) into the repo's first HTTP-streaming surface,
preserving the two-hop key posture (the Next proxy in 22-08 injects X-API-Key server-side).

Output: emit_ui_spec tool + capability-gated offering, the SSE stream + regenerate endpoints,
main registration, unit tests.
</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Client → POST /v1/chat/stream | untrusted request body over the X-API-Key gate; the Next proxy (22-08) injects the key server-side |
| LLM tool-call output → client | emit_ui_spec spec JSON is untrusted; validated at the web boundary, not here (FOUND-6) |
| Long-lived SSE connection | an open stream consumes a worker until terminal/disconnect |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-24 | Spoofing | unauthenticated stream access | mitigate | require_api_key on the router; fail-closed 401; key never NEXT_PUBLIC_ (proxy injects it, 22-08) |
| T-22-25 | Tampering | malicious genui spec from the model | mitigate | spec passed through verbatim; safeParse→SAFE_FALLBACK_SPEC enforced at the web boundary (22-09, FOUND-6); no server-side execution |
| T-22-26 | Denial of service | never-ending / abandoned streams | mitigate | cost breaker caps turn length (22-04/06); disconnect detection cancels the run; SSE no-buffer headers |
| T-22-27 | Repudiation | disconnect losing the in-flight response | mitigate | disconnect → agent persists the partial (stopped) with a terminal run event (D-15/D-25) |
| T-22-28 | Elevation of privilege | non-genui model coerced to emit UI/tools | mitigate | emit_ui_spec offered only when the registry flags the model genui-capable (D-05); no other tools offered (D-03) |
</threat_model>

## Trust Boundaries

| storage listing → the DOM | File NAMES are attacker-controlled strings (the attacker being anyone who can put a file in the user's vault — including the user's own future sync integrations). They are rendered as text. |
| signed URL → navigation | The download URL is minted server-side but is followed by the browser. |

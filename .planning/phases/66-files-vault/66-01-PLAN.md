`client.storage.from(bucket)` yields these; every call resolves `{ data, error }`, NEVER throws:

```ts
// .list(prefix, { limit, offset, sortBy })  ->  { data: FileObject[] | null, error: StorageError | null }
type FileObject = {
  name: string;                 // basename only, NOT the full key
  id: string | null;            // ← null MEANS THIS IS A FOLDER. This is the whole folder detection.
  updated_at: string | null;    // ISO
  created_at: string | null;
  metadata: {                   // ← null on folders
    size: number;
    mimetype: string;
  } | null;
};
// .createSignedUrl(path, expiresIn, { download })   -> { data: { signedUrl: string } | null, error }
// .createSignedUploadUrl(path)                      -> { data: { signedUrl: string; token: string; path: string } | null, error }
// .upload(path, body, { contentType, upsert })      -> { data: { path: string } | null, error }
// .remove(paths: string[])                          -> { data: FileObject[] | null, error }
//   ⚠ `remove` takes EXACT keys. There is no prefix/recursive delete. Folder delete = walk + remove.
```

§B — `VaultStorageClient` — the injected seam you DEFINE in `vault-types.ts`. Declare ONLY the five
methods above, structurally. This is why the tests need no Supabase and no network, and why
`@supabase/supabase-js` is imported in exactly one place (Plan 02's router), never here:

```ts
export type RawFileObject = {
  name: string;
  id: string | null;
  updated_at: string | null;
  metadata: { size: number; mimetype: string } | null;
};

export type VaultStorageClient = {
  list(prefix: string, opts: { limit: number; offset: number; sortBy: { column: string; order: string } }):
    Promise<{ data: RawFileObject[] | null; error: { message: string } | null }>;
  createSignedUrl(path: string, expiresIn: number, opts?: { download?: string }):
    Promise<{ data: { signedUrl: string } | null; error: { message: string } | null }>;
  createSignedUploadUrl(path: string):
    Promise<{ data: { signedUrl: string; token: string; path: string } | null; error: { message: string } | null }>;
  upload(path: string, body: ArrayBuffer | Blob | Uint8Array, opts?: { contentType?: string; upsert?: boolean }):
    Promise<{ data: { path: string } | null; error: { message: string } | null }>;
  remove(paths: string[]):
    Promise<{ data: unknown; error: { message: string } | null }>;
};
```

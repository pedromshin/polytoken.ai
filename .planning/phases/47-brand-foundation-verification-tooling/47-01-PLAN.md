## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| static SVG asset → browser | `icon.svg` and the inline mark render in the user's browser; SVG can carry active content |
| brand copy → rendered DOM | login/sidebar strings render as React text nodes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-47-01 | Tampering | apps/web/src/app/icon.svg | mitigate | Asset is inert — no `<script>`, `<foreignObject>`, or event-handler attributes (grep-gated in Task 1 acceptance) |
| T-47-02 | Information disclosure | brand-mark.tsx / login copy | accept | Copy is static literals with no user/secret data; React auto-escapes text nodes |
| T-47-03 | Tampering | brand-mark.tsx color values | mitigate | No raw hex in TSX (grep-gated); colors flow from `currentColor`/tokens, so a token swap can't be bypassed by a hardcoded brand color |
</threat_model>
